Bitcoin Blockchain Data

Shown are visualizations of various aspects of the Bitcoin blockchain, where general performance, protocol adherence and evolution are shown over it's lifespan thus far.

  • The dataset can be found on Kaggle or Github.
  • An example of data visualizations can be found on Kaggle.
  • The full source code for extracting and visualizing the dataset can be found on Github.

Time To Mine Each Bitcoin Block

  • The time to mine each Bitcoin block, with a dashed black line showing the target time and a solid black line showing the rolling average of the previous 2016 blocks.
  • Target Time: 600 seconds / 10 minutes
